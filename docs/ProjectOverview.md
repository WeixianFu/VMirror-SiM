## Prompt #1：项目总体框架搭建

**日期：** 2026-03-21

**想法：** 创建一个参数化的 Blender 拖车后视镜视野仿真系统，支持多车型、多镜型、多房车尺寸、多眼点位置的组合，用于跨平台通用方法论研究和技术学报撰写。

**Prompt：**

你是一个 Blender Python 专家。请帮我在 Blender 中搭建一个"拖车后视镜视野仿真"系统。整个项目使用参数化简化模型（不需要外饰细节，用 box/基础几何体表达关键尺寸即可），通过 Cycles 渲染引擎的真实曲面反射来模拟后视镜视野。

### 一、参数化车辆模型生成

创建一个 Python 函数，根据以下参数生成简化车辆模型：
- 车身：长、宽、高、前悬、后悬、轮距
- 后视镜：安装位置（x/y/z）、镜面尺寸（宽/高）、镜面曲率半径（平面镜=∞）、镜面倾斜角度
- H点（臀点）位置：x/y/z
- 牵引连接点位置：x/y/z

预设 4 种车型参数配置：
1. 皮卡（如 Ford Ranger / Toyota Hilux 级别）
2. SUV（如中型 SUV）
3. 旅行车（如 Passat Variant 级别）
4. A级小车（如 Polo / Yaris 级别）

### 二、参数化房车模型生成

创建函数生成简化房车模型，参数包括：
- 车身：长、宽、高
- 连接杆长度
- 轮轴位置

预设 3 种尺寸：
1. 紧凑型：长≈5m，宽≈2.1m，高≈2.5m
2. 中型：长≈6.5m，宽≈2.3m，高≈2.7m
3. 大型：长≈8m，宽≈2.5m，高≈2.8m

### 三、后视镜系统

支持 4 种后视镜类型，每种用不同的几何体和材质表达：
1. **普通平面镜**：平面反射镜面，标准尺寸
2. **吸附式小凸面镜**：在普通镜面上附加一个小圆形凸面镜（给定曲率半径和直径）
3. **外套式外壳镜**：在普通镜外壳外面套一个更大的镜面（给定额外尺寸和曲率）
4. **电动升降高端镜**：双镜头设计，主镜+下方可调副镜，副镜可设定上下位移量

所有镜面材质使用 Cycles 中的 Glossy BSDF（roughness=0），确保真实反射。凸面镜使用对应曲率的球面几何体。

### 四、眼点摄像机系统

基于 H点位置，按人体工学百分位设置眼点摄像机：
- 5th 百分位（小个子，如亚洲女性）
- 50th 百分位（中等体格）
- 95th 百分位（大个子，如北欧男性）

眼点相对 H点的偏移量参考 SAE J941 标准。摄像机朝前方放置，视角模拟人眼（水平FOV≈120°）。

### 五、场景组装与渲染

创建一个主控函数，输入参数组合（车型、镜型、房车尺寸、眼点百分位），自动生成完整场景并渲染以下视图：
1. **驾驶员第一人称视角**（含后视镜反射画面）
2. **左/右后视镜特写**（单独摄像机对准镜面渲染）
3. **俯视可视区域图**（从正上方渲染，标注可视/盲区范围）
4. **光路示意图**（可视化眼点→镜面→地面的反射光路）
5. **3D 可视锥体**（半透明锥体表示后视镜可视体积）

渲染设置：Cycles，Metal GPU，分辨率 1920x1080，samples 128。

### 六、批量仿真与输出

创建批量运行脚本，遍历指定的参数组合矩阵，自动渲染并按规范命名保存图片（如 `pickup_convex_medium_P50_driver_view.png`）。输出结果汇总为一个 CSV 文件，记录每组参数对应的可视范围数据。

项目文件结构建议：
```
mirror_sim/
├── config/           # 参数配置文件（JSON/YAML）
├── scripts/          # Blender Python 脚本
├── models/           # 导入的参考模型
├── renders/          # 渲染输出
└── results/          # 分析结果（CSV等）
```
