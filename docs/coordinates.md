# 坐标系统规范

> 本文档是项目坐标系的**唯一权威参考**。所有模型原点、方向约定、变换关系均以此为准。
>
> **配置文件**：`config/vehicles.yaml`、`config/caravans.yaml`、`config/mirrors.yaml`
> **构建代码**：`src/mirror_builder.py`

---

## 一、全局坐标系

项目统一使用 **Blender 原生坐标系**，全局原点 = 场景原点 = 车辆前轴中心地面投影。

| Blender 轴 | 方向 | 汽车工程含义 | 正方向 |
|-----------|------|------------|--------|
| **X** | 左右 | 车辆横向 (Lateral) | **正 = 副驾侧 (右)** |
| **Y** | 前后 | 车辆纵向 (Longitudinal) | **正 = 车头 (前进方向)** |
| **Z** | 上下 | 车辆垂直 (Vertical) | **正 = 向上** |

**单位**：米 (m)

**方向常量**（代码中使用）：
```
FORWARD = "+Y"    LEFT  = "-X"    UP   = "+Z"
REAR    = "-Y"    RIGHT = "+X"    DOWN = "-Z"
```

## 二、旋转约定

### 主用：Blender Euler XYZ

| 分量 | 含义 | 绕哪个轴 |
|------|------|---------|
| rx | 绕 X 轴旋转 | 横向轴 |
| ry | 绕 Y 轴旋转 | 纵向轴 |
| rz | 绕 Z 轴旋转 | 垂直轴 |

### 汽车工程映射 (SAE J670)

| 汽车术语 | 物理含义 | 对应 Blender | Euler 分量 |
|---------|---------|-------------|-----------|
| **Yaw** 偏航 | 车头左右转向 | 绕 Z 轴 | rz |
| **Pitch** 俯仰 | 车头抬起/低头 | 绕 X 轴 | rx |
| **Roll** 侧倾 | 车身左右倾斜 | 绕 Y 轴 | ry |

> **注**：大多数物体旋转为 (0, 0, 0)。后视镜旋转由 `calculate_mirror_orientation()` 根据反射定律动态计算，不存储固定值。

## 四、房车本地坐标系

### 原点
**球头挂接点 (50mm ball coupler)**，离地 0.42m

- 原点在牵引杆最前端的挂接位置
- 放置房车时，将原点对齐车辆 hitch 点即可
- +Y = 牵引前进方向（与车辆一致）
- 地面在本地 Z = -0.42m

### 各尺寸 Bounding Box

| 尺寸 | 参考型号 | config 宽 | config 总长 | 牵引杆 | 模型宽 | 模型总长 | bbox Z |
|------|---------|----------|-----------|--------|--------|---------|--------|
| S (紧凑) | Eriba Touring | 2.08m | 5.0m | 1.2m | 2.080m | 5.546m | [-0.42, +1.87] |
| M (中型) | Hobby De Luxe | 2.30m | 6.5m | 1.2m | 2.300m | 6.525m | [-0.42, +2.59] |
| L (大型) | Hobby Prestige | 2.50m | 8.0m | 1.35m | 2.500m | 6.876m | [-0.42, +2.43] |

> **模型朝向**：所有房车 .blend 文件中，牵引杆 (drawbar) 朝 +Y 方向，车身向 -Y 延伸。挂接点在 Y=0。

### 挂接几何

| 参数 | 值 | 说明 |
|------|-----|------|
| hitch_height | 0.42m | 标准 50mm 球头离地高度 |

## 七、变换链

### 层级关系

```
Global (Scene)
├── Vehicle (原点 = 全局原点, 位置=(0,0,0), 旋转=(0,0,0))
│   ├── Mirror_Assembly_L (Empty, 位于安装点)
│   │   └── Mirror_Glass_L (位于安装点+镜臂偏移, 旋转=动态计算)
│   └── Mirror_Assembly_R
│       └── Mirror_Glass_R
└── Caravan (原点=球头挂接点, 位置=(0, hitch_y, 0), 旋转=(0,0,0))
```

### 变换公式

**镜面中心世界坐标**：
```python
# 安装点（从 vehicles.yaml 读取）
mount = vehicles_config[vehicle_key]["mirror_mount"][side]  # (x, y, z)

# 镜臂偏移（从 mirrors.yaml 读取）
offsets = mirrors_config["mirror_offsets"][offset_category]
lateral_sign = -1 if side == "left" else +1

glass_world = (
    mount[0] + lateral_sign * offsets["arm_length"],
    mount[1] + offsets["arm_forward"],
    mount[2] + offsets["glass_height"]
)
```

**镜面旋转**：
```python
# 由反射定律动态计算，不存储固定值
euler = calculate_mirror_orientation(glass_center, eye_point, target_point)
# 参见 src/mirror_builder.py:calculate_mirror_orientation()
```

**房车全局位置**：
```python
caravan_origin_global = (0, hitch_y, 0)
# hitch_y = -(overall_length - front_overhang) - rear_bumper_to_hitch
```

### 计算示例：CR-V + standard_convex + 左镜

```
安装点:    (-0.913, -0.45, 1.25)
偏移类型:  standard (arm_length=0.15, arm_forward=0.02, glass_height=0.05)
lateral_sign = -1 (left)

镜面中心:  (-0.913 + (-1)*0.15, -0.45 + 0.02, 1.25 + 0.05)
         = (-1.063, -0.43, 1.30)

旋转:      calculate_mirror_orientation((-1.063,-0.43,1.30), (-0.35,-0.10,1.20), (0,-20,0.5))
```

### 旋转

镜面旋转不存储固定值，由 `src/mirror_builder.py:calculate_mirror_orientation()` 根据以下参数动态计算：
- `glass_center`：镜面中心世界坐标（上表）
- `eye_point`：驾驶员眼点（第三节）
- `target_point`：目标点（默认 `(0, -20, 0.5)`，车正后方 20m 离地 0.5m）

### 验证标准

镜面位置必须满足（`src/mirror_builder.py:validate_mirror_position()`）：
- 镜面 X 在车身外侧（|X| > body_width/2）
- 镜面高度 0.8m < Z < 1.8m
- 镜面纵向 -1.0m < Y < 0.2m
- 法向量与眼点方向夹角 < 60°
