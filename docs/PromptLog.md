# AI Prompt 记录

> 本文件记录项目开发过程中使用的 AI Prompt 及其执行结果，作为开发日志留存。

---

## Prompt #1：项目总体框架搭建

**日期：** 2026-03-21

**想法：** 创建一个参数化的 Blender 拖车后视镜视野仿真系统，支持多车型、多镜型、多房车尺寸、多眼点位置的组合，用于跨平台通用方法论研究和技术学报撰写。

**执行结果：** 建立了初始目录结构和 README。后续各模块的设计细节分别记录在对应的文档中（VehicleModels.md、CaravanModels.md、MirrorDesign.md、TestScene.md）。

---

## Prompt #2：后视镜精确建模与定位

**日期：** 2026-03-25

**想法：** 解决通过 Blender MCP 构建后视镜时"位置不对、尺寸不对、角度不对、形状不对"的系统性问题。基于汽车工程坐标系、真实车型参考数据、反射定律，构建一套可靠的后视镜建模流程。

**文件：** `MirrorDesign.md`

---

## Prompt #3：AI 3D 车辆模型生成

**日期：** 2026-03-25

**想法：** 为 4 种车型（SUV / 皮卡 / 旅行车 / A级小车）创建 AI 3D 模型生成 Prompt，包含统一风格前缀、分区描述、后视镜排除指令、生成后校准要点。

**文件：** `VehicleModels.md`

---

## Prompt #4：AI 3D 房车模型生成

**日期：** 2026-03-26

**想法：** 为 3 种尺寸房车（S / M / L）创建 AI 3D 模型生成 Prompt，包含连接几何、宽度遮挡分析、坐标系对齐公式。

**文件：** `CaravanModels.md`

---

## Prompt #5：测试场景搭建

**日期：** 2026-03-26

**想法：** 搭建标准化外部测试场景，包含道路系统、车道线、参考标记物，用于验证镜面反射效果和量化可视区域。

**文件：** `TestScene.md`
