# 坐标计算推导

> 本文档从 Blender 模型实测数据出发，逐步推导后视镜的位置、朝向和可视范围。
> 以 CR-V (SUV) 为算例，走通后推广到其他车型。

---

## 0. 符号约定

### 坐标系

采用 Blender 原生坐标系，全局原点 = 车辆前轴中心地面投影。

| 轴 | 方向 | 正方向 |
|----|------|--------|
| X | 横向 | 副驾侧（右） |
| Y | 纵向 | 车头（前进方向） |
| Z | 垂直 | 向上 |

### 符号表

| 符号 | 含义 | 单位 |
|------|------|------|
| $W$ | 车身宽度（不含后视镜） | m |
| $P_{\text{mount}}$ | 后视镜安装基准点 | (x, y, z) m |
| $P_{\text{glass}}$ | 镜面中心世界坐标 | (x, y, z) m |
| $P_{\text{eye}}$ | 驾驶员眼点 | (x, y, z) m |
| $P_{\text{target}}$ | 目标点（希望在镜中看到的点） | (x, y, z) m |
| $\hat{n}$ | 镜面法向量（单位向量） | - |
| $R$ | 凸面镜曲率半径 | m |
| $\theta_{\text{FOV}}$ | 视场角 | rad / ° |

### 1.3 镜臂偏移 → 镜面中心

镜面中心 = 安装点 + 镜臂偏移向量。镜臂将镜面向三个方向偏移：

| 偏移分量 | 方向 | 含义 |
|----------|------|------|
| `arm_length` | 向车外侧 (±X) | 镜面伸出车身 |
| `arm_forward` | 向前 (+Y) | 镜面略微前倾 |
| `glass_height` | 向上 (+Z) | 镜面比安装点略高 |

公式：

$$P_{\text{glass}} = P_{\text{mount}} + \begin{pmatrix} s \cdot L_{\text{arm}} \\ d_{\text{fwd}} \\ d_{\text{up}} \end{pmatrix}$$

其中 $s = -1$（左侧）或 $s = +1$（右侧）。

镜臂偏移参数（基于实车数据和 ECE R46 法规重新计算）：

**设计依据**：
- SUV 标准镜面超出车身约 100mm（[实车数据](https://www.elementownersclub.com/threads/width-of-side-view-mirrors.12439/)）
- 拖车镜需看过最宽房车（2.5m），超出车身约 370mm
- 广角副镜/盲点镜位于主镜下方或外下角
- 驾驶员眼点估计：$P_{\text{eye}} = (-0.35, -1.0, 1.40)$

**目标点（基于 [ECE R46 Class III](https://eur-lex.europa.eu/legal-content/EN/TXT/HTML/?uri=CELEX:42014X0808(02))）**：
- 主镜（standard / towing_main / electric_main）：左 (−2.5, −15, 0.5)，右 (+3.0, −20, 0.5)
- 广角副镜（towing_wide_angle / electric_sub）：左 (−1.5, −4, 0.3)，右 (+1.5, −4, 0.3)
- 盲点镜（clip_on）：左 (−1.5, −5, 0.5)，右 (+1.5, −5, 0.5)

## 1.5 车辆-房车位置关系

### Hitch 地面投影点（车辆参考点）

房车通过拖车钩（hitch）连接到车辆后方。为保证车辆和房车都在地面上（Z=0），使用 **hitch 球头在地面的投影点**作为车辆参考点。

$$P_{\text{hitch\_ground}} = (0,\ Y_{\text{rear\_bumper}} - 0.10,\ 0)$$

其中 $Y_{\text{rear\_bumper}}$ 为模型后保险杠 Y 坐标（bbox Y_min），0.10m 为拖车钩在后保后方的典型偏移量。

#### 各车型 Hitch 地面投影点

| 车型 | 后保 Y (m) | Hitch 地面投影 (x, y, z) |
|------|-----------|------------------------|
| CR-V (SUV) | −3.560 | (0, −3.660, 0) |
| Hilux (Pickup) | −3.329 | (0, −3.429, 0) |
| Passat (Wagon) | −3.860 | (0, −3.960, 0) |
| Polo (Hatchback) | −3.587 | (0, −3.687, 0) |

### 房车相对于 Hitch 地面投影点的偏移

所有房车尺寸（S/M/L）使用相同偏移：

$$P_{\text{caravan}} = P_{\text{hitch\_ground}} + (0,\ +0.050,\ +0.420)$$

| 分量 | 偏移 | 说明 |
|------|------|------|
| ΔX | 0 | 中线对齐 |
| ΔY | +0.050m | 略向前（牵引杆间距微调） |
| ΔZ | +0.420m | 球头挂接高度（房车原点在 hitch ball 处） |

### 房车全局坐标计算

```
房车全局位置 = 车辆全局位置 + Hitch 地面投影偏移 + 房车相对偏移
```

#### 各车型-房车组合的房车放置坐标

| 车型 | Hitch 地面投影 | 房车放置位置 (x, y, z) |
|------|--------------|----------------------|
| CR-V | (0, −3.660, 0) | (0, −3.610, 0.420) |
| Hilux | (0, −3.429, 0) | (0, −3.379, 0.420) |
| Passat | (0, −3.960, 0) | (0, −3.910, 0.420) |
| Polo | (0, −3.687, 0) | (0, −3.637, 0.420) |

> 房车放置位置 = hitch 地面投影 + (0, +0.05, +0.42)，适用于 S/M/L 所有尺寸。

## 2. 镜面朝向推导

### 2.1 真实后视镜朝向参考

网络检索结果（[SAE BGE 方法](https://99percentinvisible.org/article/seeing-blind-spots-clever-trick-properly-align-cars-side-view-mirrors/)、[FMVSS 111](https://www.law.cornell.edu/cfr/text/49/571.111)、[ECE R46 Rev.6](https://www.unece.org/fileadmin/DAM/trans/main/wp29/wp29regs/2016/R046r6e.pdf)）：

| 参数 | 典型范围 | 来源 |
|------|---------|------|
| 镜面法向量偏航角（Yaw） | 15°~ 25°（偏离正后方） | SAE BGE / 工程测量 |
| 镜面法向量俯仰角（Pitch） | ±10° 以内，SUV 略下倾 | 调整经验 / 专利文献 |
| 最小水平 FOV | ≥ 20° | FMVSS 111 |
| 凸面镜对称轴倾角 | ≈ 8.5° | 专利 US4629296 |

> ECE R46 定义**最低视野覆盖范围**（地面区域），而非固定的镜面角度。实际角度取决于车辆几何和驾驶员眼点位置。

### 2.2 反射定律向量形式

**问题**：已知镜面中心 $P_{\text{glass}}$、驾驶员眼点 $P_{\text{eye}}$、目标点 $P_{\text{target}}$，求镜面法向量 $\hat{n}$，使驾驶员通过镜面能看到目标点。

**反射定律**：入射角 = 反射角。即入射光线与法向量的夹角等于反射光线与法向量的夹角。

设：
$$\hat{d}_{\text{eye}} = \frac{P_{\text{eye}} - P_{\text{glass}}}{|P_{\text{eye}} - P_{\text{glass}}|}$$
$$\hat{d}_{\text{target}} = \frac{P_{\text{target}} - P_{\text{glass}}}{|P_{\text{target}} - P_{\text{glass}}|}$$

由反射定律，法向量是两个方向的**角平分线**：

$$\hat{n} = \frac{\hat{d}_{\text{eye}} + \hat{d}_{\text{target}}}{|\hat{d}_{\text{eye}} + \hat{d}_{\text{target}}|}$$

**直觉理解**：镜面法向量恰好在"看向眼睛"和"看向目标"两个方向的正中间，这样反射定律自然满足。

### 2.3 法向量 → 欧拉角

Blender 中平面的默认法向量为 $\hat{z} = (0, 0, 1)$（朝上）。需要计算从 $\hat{z}$ 旋转到 $\hat{n}$ 的旋转角。

步骤：
1. 计算四元数 $q$ 使得 $q \cdot \hat{z} = \hat{n}$（使用 `Vector.rotation_difference()`）
2. 将四元数转换为 XYZ 欧拉角

```python
default_normal = Vector((0, 0, 1))
rotation = default_normal.rotation_difference(mirror_normal)
euler = rotation.to_euler("XYZ")
```

### 2.5 CR-V 数值算例（standard_convex 左镜）

```
P_glass  = (-0.955, -0.695, 1.369)
P_eye    = (-0.35,  -0.10,  1.20)     ← 待实测替换
P_target = ( 0,     -20,    0.50)

Step 1: 方向向量
  d_eye    = P_eye - P_glass    = (0.605, 0.595, -0.169)
  |d_eye|  = 0.867
  d̂_eye    = (0.698, 0.686, -0.195)

  d_target = P_target - P_glass = (0.955, -19.305, -0.869)
  |d_target| = 19.334
  d̂_target = (0.049, -0.999, -0.045)

Step 2: 法向量（角平分线）
  sum      = d̂_eye + d̂_target = (0.747, -0.313, -0.240)
  |sum|    = 0.849
  n̂        = (0.880, -0.369, -0.283)

Step 3: 验证角度
  与 -Y 轴（正后方）的偏航角 ≈ arctan(0.880/0.369) ≈ 67°
  → 这是法向量方向，不是视线方向。
  法向量与正后方的夹角 ≈ arccos(0.369) ≈ 68°
```

> ⚠ 以上数值基于估计的眼点，仅用于验证公式逻辑。需实测眼点后重新计算。
